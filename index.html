Web Service Resep Masakan

Untuk memenuhi salah satu syarat ujian tengah semester dalam mata kuliah Praktik Pemrograman Multi Platform, saya bersama rekan project membangun sebuah Web Service Rest-API untuk Resep Masakan menggunakan Django Rest Framework. Project ini bertujuan untuk mengelola data dan mengimplementasikan REST API. Kami juga akan melakukan pengujian menyeluruh terhadap API ini menggunakan Insomnia untuk memastikan semua fungsionalitas berjalan dengan baik.

Aplikasi Mobile Pencarian Data Buku

Ini merupakan project Aplikasi Mobile Pencarian Data Buku yang saya kembangkan sebagai salah satu tugas dalam mata kuliah Pemrograman Mobile. Aplikasi ini memungkinkan pengguna untuk mencari informasi buku dengan mudah menggunakan kata kunci. Dengan antarmuka yang intuitif, pengguna dapat memasukkan kata kunci tertentu untuk menemukan data buku yang relevan secara cepat dan efisien. Aplikasi ini dikembangkan menggunakan bahasa pemrograman Dart untuk antarmuka mobile, PHP Native untuk backend, dan database MySQL untuk penyimpanan data.

Aplikasi Mobile CRUD Data Tiket Konser

Project ini merupakan Aplikasi Mobile CRUD Data Tiket Konser yang dikembangkan sebagai syarat untuk ujian akhir semester dalam mata kuliah Pemrograman Mobile. Aplikasi ini memungkinkan pengguna untuk membuat, membaca, memperbarui, dan menghapus data tiket konser. Dengan fitur-fitur yang lengkap dan antarmuka yang user-friendly, pengguna dapat dengan mudah mengelola informasi tiket konser, seperti detail acara, harga, dan ketersediaan tiket. Aplikasi ini dibangun menggunakan bahasa pemrograman Dart untuk antarmuka mobile, Native PHP untuk backend, dan database MySQL untuk penyimpanan data.

Aplikasi Mobile Comb Sorting
